######################
EXAMPLE COMMAND:
######################
1.compile
java ././*.java

2.summarize
      java .\summarize.java .\test_video\ .\test_video.wav 2    
OR    java .\summarize.java .\test_video_2\ .\test_video_2.wav 0

3.Display
      java .\videoDisplay.java .\test_videosummarize\ .\test_videoAudio.wav 
######################


Instructions:
1. compile all the files:
 javac ././*.java

2. Summarize the video:
 java summarize.java [RgbFolderName] [AudioFileName] [SummarizeOffset(0/1/2)]
      eg: java summarize.java .\concert\ .\concert.wav 0

After running the summarize function, there will be two contents add to the root directory.
One is the directory named [RgbFolderName]+summarize, which saved all rgbFrames in the summarized video.
The other one is the audio file named [AudioFileName]+summarize.wav.

3. Display the video:
 java videoDisplay.java [RgbOutputFolderName] [AudioOutputFileName]
      eg: java videoDisplay.java .\concertsummarize\ .\concertAudio.wav
This time, the RgbFolderName and AudioFileName are the files generated by the summarize.java.
The foldername should be original foldername added with "summarized" such as concertsummarize. The audio file name should be original name + Audio such as concertAudio.wav
